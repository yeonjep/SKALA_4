# SKALA_4

SKALA 4기 교육 과정에서 진행한 실습, 과제, 프로젝트를 정리하는 저장소입니다.

교육 기간 동안 학습한 내용을 코드로 구현하고, Git을 활용하여 버전 관리 및 기록합니다.

---

## 📌 About

이 저장소에는 교육 과정에서 수행한 다양한 실습과 과제가 포함됩니다.

- Python 프로그래밍
- 자료구조 및 알고리즘
- 데이터 처리 및 분석
- 웹 개발
- 프로젝트 실습
- 기타 학습 내용

---

## 🛠 Tech Stack

- Python
- Git / GitHub
- VS Code

> 교육 과정에 따라 사용 기술은 계속 추가될 예정입니다.

교육 과정이 진행됨에 따라 실습 및 프로젝트가 주차별 또는 주제별로 추가됩니다.

